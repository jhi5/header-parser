J. Munford's Who Am I API

How To Use

Either click the button above or navigate to

http://jmunford-header-parser.herokuapp.com/whoami

And you'll get a JSON object back with your IP, operating system and language setting! It looks something like this

{"ipaddress":"localhost:3000","language":"en-US,en;q=0.8","software":"X11; Linux x86_64"}
